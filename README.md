**DESCRIPCION:
              Este proyecto trata sobre el desarrollo de una pagina web estatica hecha con HTML.
              La plataforma cuenta con varias paginas conectadas entre si, utilizando una estructura semantica para organizar mejor el 
              contenido y facilitar la navegacion.
              
Paginas Desarrolladas:
              *Index.html: Pagina principal donde se muestra un resumen con estadisticas y novedades.
              *Proyectos.html: Sección donde se pueden ver distintos proyectos junto con filtros de búsqueda.
              *Detalles.html: Página donde se muestra información más detallada de cada proyecto.
              *Perfil.html: Seccion con los datos del alumno y profesor

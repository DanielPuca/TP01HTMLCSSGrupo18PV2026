**DESCRIPCION:

      Este proyecto trata sobre el desarrollo de una pagina web estatica hecha con HTML.
      La plataforma cuenta con varias paginas conectadas entre si, utilizando una estructura 
      semantica para organizar mejor el contenido y facilitar la navegacion.
              
Paginas Desarrolladas:

      *Index.html: Pagina principal donde se muestra un resumen con estadisticas y novedades.
      *Proyectos.html: Sección donde se pueden ver distintos proyectos junto con filtros de búsqueda.
      *Detalles.html: Página donde se muestra información más detallada de cada proyecto.
      *Perfil.html: Seccion con los datos del alumno y profesor

              
.....Parte 2 CSS.......

Se utilizó CSS para dar diseño y estructura a las páginas web, aplicando diferentes conceptos como:

  * Variables CSS (:root) para definir colores y mantener un estilo uniforme en todo el sitio.
  * Flexbox para alinear y distribuir elementos como el menú de navegación y el contenido.
  * CSS Grid para organizar los elementos en forma de grilla adaptable.
  * Efectos de interacción como :hover en botones y enlaces.
  * Estilos visuales como bordes redondeados (border-radius), sombras (box-shadow) y espaciados (margin, padding).
  * Uso de imágenes adaptables con object-fit y diseño de avatares circulares en perfiles.
